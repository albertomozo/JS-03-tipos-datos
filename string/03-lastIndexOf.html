lastIndexOf()

El método indexOf() devuelve la posición de la última aparición de un valor especificado en una cadena. Checkea o comprueba la cadena de texto al revés del indexOf()
Este método devuelve -1 si el valor de búsqueda núnca ocurre. Si no está, sale -1.
Es sensible a mayúsculas.
El índice del primer carácter es el 0, el segundo es 1 y así sucesivamente.
*** sirve para validar formularios, o emails (si tiene o no tiene @, para teléfonos...) = textos
La posición siempre es la misma = 0 1 2 3 4 5 6 .... núnca se altera.
